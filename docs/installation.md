# Installation

## Stable release

To install DS Git Homework, run this command in your terminal:

```sh
uv add ds_git_homework
```

Or if you prefer to use `pip`:

```sh
pip install ds_git_homework
```

## From source

The source files for DS Git Homework can be downloaded from the [Github repo](https://github.com/safffka/ds_git_homework).

You can either clone the public repository:

```sh
git clone git://github.com/safffka/ds_git_homework
```

Or download the [tarball](https://github.com/safffka/ds_git_homework/tarball/master):

```sh
curl -OJL https://github.com/safffka/ds_git_homework/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd ds_git_homework
uv pip install .
```
